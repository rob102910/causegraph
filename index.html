Bar Chart of Color

For this lab, we found out that the code that draws the two diagrams totally overlaps.
We can just use one single function called drawChart( ) which will take the data from json file or csv file as input to draw the two diagrams.
Here are the pesudo code for how to draw the diagrams using different kinds of files.

First use the d3 to select the data:
var dataP=d3.json/csv("colordata.json/csv");

Then use the data we select to call the drawChart( ) function:
dataP.then(function(data)
{
drawChart(data);
},
function(err)
{
console.log(err);
})
We only need to change the var dataP to draw two diagrams from different kinds of files. And you can take a closer look at our js file.

In order to distinguish the difference between the graph created by json file and csv file, we made a little change to the data.
As you can see by clicking the links for those two data, the json file has blue as the first color, but csv file has red as the first color.
